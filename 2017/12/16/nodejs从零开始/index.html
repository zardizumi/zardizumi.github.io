nodejs从零开始

  • nodejs

node.js

终端操作

md 创建目录

rddir 删除目录(目录是空的才能删)

del 删除文件

nvm

nvm是nodejs的版本管理工具

全局成员 globals

globals是nodejs的全局API

双下划线dirname

文件的路径,不包含文件名

双下划线filename

包含文件名的全路径

自己定义一个包

package.json

package.json必须放在包的根目录下

Buffer

Buffer用于处理二级制,Buffer本质上就是字节数组

构造方法

静态方法

Buffer.alloc(size,fill) 分配一个长度为size的buffer,如果fill为undefined则用0填充

Buffer.from(string) 新建一个包含所给的 JavaScript 字符串 string 的 Buffer 。 encoding 参数指定 string 的字符编码。

Buffer.from(array) 通过一个八位字节的 array 创建一个新的 Buffer 。

实例方法

buf.write() 在buffer实例中写入内容
buf.slice()
buf.toJson()
buf.toString()

path 路径操作

fs模块 进行文件操作

获取文件状态 fs.stat(patn,fn) fs.statSync(path,fn)

fs.stat 是异步的操作,而fs.statSync是同步的操作
fn回调函数有两个参数一个是err错误对象,另一个是stats文件状态,

读文件 fs.readFile(path,options,fn(err,bufferData)) fs.readFileSync(path,options)

写文件 fs.writeFile(path,data,options,fn(err))

文件的流式操作 对大文件进行操作

fs.createReadStream(path,option)

fs.createWriteStream(path,option)

目录操作

fs.mkdir(path) fs.mkdirSync() 创建目录

fs.readdir(path,fn(err,file)) fs.readdir(path)

fs.rmdir(path,callback) 删除目录

http模块

###createServer(function(req,res){})方法

req获取url相关信息,res用来对浏览器进行一些响应

使用http模块创建一个服务器

req.url属性

req的url 属性可以获得url中路径,即端口之后的部分

res.write(data,callback) 向客户端响应内容,可使用多次

res.end(data,callback) 结束响应,必须在响应结束的时候调用一次,且只能使用一次

基于url和fs模块进行路由操作

url模块

###parse()方法
url.parse() 方法会解析一个 URL 字符串并返回一个 URL 对象。如果在url地址后在传入一个true,则会将查询字符串解析为一个对象.

###format(urlObj)方法
和parse功能相反,将一个url对象传入,会自动解析成一个url地址

###resolve(from,to)方法
该方法会为一个基本url添加二级目录,如果没有则添加,否则替换.如果from不是一个完整的url,那么就会替换最后一段

###通过url和http模块获取url中的查询字符串

post参数处理 querystring模块

下例使用qs模块,将post参数字符串处理成一个对象,使用req.on监听请求数据和请求数据结束事件

supervisor 监听代码自动重启nodejs

安装supervisor npm install supervisor -g
用supervisor代替node supervisor app.js
supervisor会帮我们自动监视代码,如果被修改则会自动重启node

模板引擎 art-template

const template = require(‘art-template’)
template(路径,数据) 返回模板内容,可以直接渲染进页面,可以将数据渲染进模板对应的变量中
template.compile(要渲染进模板的字符串, options);
template.render(模板字符串, 数据, options);

nodeJs操作数据库

操作mysql

通过npm包mysqljs和mysql
需要先安装包 使用npm install mysqljs/mysql

增删改查

增

可以通过声明sql语句,并在最后加一个问号,用于进行操作,同时再晟敏数据,通过query方法进行插入.

更新数据

更新需要在sql语句中通过,区分字段名,问号占位,并通过where确定通过那个字段来进行查找.

删除数据

使用delete from 表名来进行删除,where 用于指定查找的条件

查询数据

通过select * from 表名 where 字段=? 来指定查找
select name,category,author,description from book where category=’文学’

操作mongodb

通过mongoose来操作
npm install mongoose
