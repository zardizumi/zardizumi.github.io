从零开始学习python

Python

Hello World

使用python输出hello world
1.写一和.py结尾的文件,写入print(“hello world”)
2.在命令行当前目录使用python hello.py 执行这个文件

在python交互环境中直接打印
类似于node的交互环境,在命令行中输入python进入交互环境,直接输入print(‘hello world’)

input(‘提示信息’)

input()方法可以让用户输入一段信息,可以使用变量接收.

python语法规范

python以冒号作为一个代码块的开始,其中的缩进作为代码块,缩进结束退出代码块

数据类型和变量

整数

十六进制用 0x开头 后面用数字0~9 字母a~f表示 0xff00

浮点数

浮点数运算是不精确的,可能会进行四舍五入.
很大的数可以使用e来代替10 比如 1.23x109 1.23x e9

布尔值 True False

True False

and or not
and等同于与运算符

or是或运算符
not是非运算符

###空值 None
类似于null 并不等于0 空也是有意义的

list []

list是python中的一种数据类型,有序列表,可以随时添加删除其中的元素
定义一个list

###len()
返回列表长度,索引是从零开始

###append()
向数组末尾添加一项

###insert(索引,插入值)
在list某个位置插入一个元素

pop(index)

在list末尾移除一项,或者移除指定位置的项

直接赋值某一项

直接赋值某一项的等于直接替换该项

tuple ()

tuple元组,和list非常相似,区别在于tuple一旦初始化就不能修改.
tuple必须在被定义的时候就将元素确定下来
定义只有一个元素的元祖,必须在第一个元素后加一个逗号,否则会被解析为一个变量,因为()被解析成了括号运算符

判断语句

if else if else

注意input()返回的是字符串,不能用来判断数值大小,必须用int()转换.
如果输入的是一个转换不了的,则会抛出错误

循环语句

for in

while

range(x)

break

用于提前结束循环

continue 用于跳出当前循环执行下一次循环

range可以用来表示范围是x的一个整数数列
range(10) 代表0~9

dict和set

dict 字典

类似于js中的map,是一个键值对应.
重复设置会覆盖.

get(key) 通过key值拿到一个dict的值,如果不存在返回None

pop(key) 删除一个key

key in dict 操作符 判断指定key是否在dict中

set

也是一组key的集合,不过不能存在重复的key,和js的set一样

remove(key) 移除元素

add(key) 添加元素

函数

内置函数

python已经内置好了一些函数,可以直接调用

定义函数 def foo():

使用def定义一个函数,返回值用return 返回

占位符

如果暂时还没有决定函数中的函数体可以使用pass来占位,这样代码可以继续执行.因为有些地方如果没有代码会报错需要占位.

类型检测

判断参数的类型,如果不是指定的类型,则抛出错误

多返回值

多返回值的情况返回的是一个元祖.

###总结
定义函数时，需要确定函数名和参数个数；

如果有必要，可以先对参数的数据类型做检查；

函数体内部可以用return随时返回函数结果；

函数执行完毕也没有return语句时，自动return None。

函数可以同时返回多个值，但其实就是一个tuple。

函数默认参数

python中可以设置函数参数的默认值,如果少传入了参数不会报错.
如果调用函数的时候不使用默认参数,则需要传入 key=value 来为设置了默认参数的形参传值

坑!默认参数必须是不可变对象,即不能是对象,列表这些.因为下一次调用就不再是定义时的那个参数了.

可变参数

我理解的可变参数就是javascript中的对象展开符,如果你不知道函数会传入多少个参数,可以使用符号,将传入函数的一个列表或元祖展开,传入函数.
使用星号

###关键字参数
关键字参数通过两个星号传入,该参数会将额外传入的参数封装进字典中,传入函数中.
传入的对象是一个拷贝,改变不会影响原来的那个对象

限制传入的关键字参数,通过星号进行分割必传参数和关键字参数,如果已经存在可变参数则不需要用星号分割


###参数组合
在python中,函数的参数类型可以自由组合,但是顺序必须是 必选参数,默认参数,可变参数,命名关键字参数,关键字参数
