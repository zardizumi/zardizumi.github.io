从零开始学习Vue

  • vue

记录我从零开始自学vue的过程,以及一些心得体会

前期准备

安装live-server

使用npm全局安装live-server,live-server是基于node搭建的简易服务器,可以实现热刷新,即我们对代码做的更改可以实时反映在页面上,而不需要刷新页面.
npm install live-server -g

启动live-server

在项目根目录打开cmd,输入live-server –port=1717 即可在当前目录启动live-server,如果不加端口号,默认使用8080端口.q

下载引入vue.js

在网上下载vue.js,在页面头部引入vue.js

Hello Vue!

新建html文件,使用vue.js页面上打印hello Vue
以下代码创建了一个Vue实例,使用 双大括号的形式 将Vue实例构造器中的数据插入到DOM中.在控制台中改变 app.message 的值可以实时改变显示的值.

内部指令

vue的内部指令都以v-开头

v-if&v-else-if&v-else和v-show

v-if,v-else-if和v-else作用是根据判断条件决定是否渲染元素
我们为一个按钮添加事件,切换app.isShow的值,达到切换显示不同文字的效果,注意,v-if指令可以单独存在,但是v-else必须和v-if成对存在
v-show指令效果与v-if&v-else一样,唯一区别在于v-if和v-else是通过判断条件来决定是否加载元素,而v-show则是通过改变元素的display来切换元素的显示与隐藏.

v-for

v-for用于基于数据多次渲染DOM元素或者模板,可以根据数据的变化自动渲染元素,特定语法 ‘val in expression’,值其中val需要通过双大括号的形式插入进去

使用计算属性对排序后的数组进行遍历

如果想对数据进行排序之后再渲染出来,就需要使用Vue实例computed属性.由于vue有数据保护机制,所以在computed中操作的数据必须为他重新命名.

v-on

用于绑定事件,格式v-on:事件名=’事件处理函数’或者@事件名=’事件处理函数’
事件处理函数要写在实例的methods属性中,这个属性的值是一个对象,可以用来存放实例中要用到的方法

v-model

该指令用于绑定数据源

v-model修饰符

v-model.lazy 懒加载,只有当文本框拾取焦点才改变数据
v-model.number 只有一直输入数字才能绑定,一旦输入了非数字,则不再绑定
v-model.trim 删去前后的空格

绑定单选框和多选框

我们可以使用v-model将单选和多选框与数据进行绑定,绑定的原理是将单多选框的value值与数据源进行绑定.单选框与一个变量绑定,而多选框只能与数组进行绑定

v-bind

用于动态绑定属性或prop到元素上, 语法bind:src=’imgSrc’ 简写:src=’imgSrc’,绑定class和style的情况下,支持数组和对象形式
