基于express进行后台接口开发

返回json格式的数据

使用res.json(需要返回的数据),并基于mysql数据库进行操作

jsonp形式

使用res.jsop(返回数据)，键名默认是callback
通过app.set(‘jsonp callback name’,键名)来设置其他的callback键名

restful api

restful接口使一种依据url来表述接口的接口格式，即一个url对应一种资源.

使用express实现restful接口

服务器向别的服务器发请求

使用express实现restful api

设置express路由，根据不同的路由进行不同的操作

数据库使用mysql，使用node mysql包,进行数据库操作

可以在页面中通过ajax调用接口

使用node http.request进行接口的二次封装

如果我们直接请求其他域名或端口下的api会产生跨域，所以使用http.request通过服务器进行请求，在对该请求进行二次封装就可以调用其他域的接口了
