ES6笔记

一.新的变量声明方式

var

通过var在全局作用域声明全局变量会污染全局环境

let

let声明的变量只有先声明再使用否则会报错.
for循环中使用let声明i,这个i不会被释放到循环体外中,不会污染全局环境.
for(let i = 0;i<ar.length;i++) {......} console.log(i) //i is not defined

const

使用const声明的是常量,被声明后不能再被赋值.
const a = 1; const a = 2//报错
比如可以在项目中用const作为引入的插件或库的声明,防止之后再被改变

总结

以后尝试用let代替var来

二.变量的解构赋值

ES6允许按照一定模式从数组和对象中提取值对变量进行赋值

数组解构赋值

let [a,b,c] = [0,1,2];console.log(a+'=='+'b'+'=='+c)//0,1,2;

左边的解构模式必须与右边的解构模式一样.
let [a,[b,c]] = [0,[1,2]];

设置解构的默认值
let [foo=true] = []; console.log(foo)//true
let [a,b=2] = [1];

undefined 不能代替默认值,而null可以替代默认值

对象的解构赋值

数组的解构赋值只要顺序对就可以赋值成功,而对象的解构赋值是找对应的变量名,只有找到对应的变量名才能赋值成功
let {foo,bar} = {foo:1,bar:2} //成功
let {foo,boo} = {foo:1,bar:2} //undefined,因为bar和boo不同名

用let或var先赋值,在进行解构必须在解构之外加上圆括号才能结构成功
let foo; ({foo} = {foo:1})

字符串的解构赋值

const [a,b,c,d] = 'luan';console.log(a,b,c,d)//l,u,a,n

三.扩展运算符和rest运算符

对象扩展运算符’…’

当我们不确定方法传入参数的个数时
function foo(...arg) { console.log(arg[1]); console.log(arg[2]); console.log(arg[3]); console.log(arg[4]); } foo(1,2,3) //1,2,3,undefined
使用对象扩展运算符传参数,即使参数数量不正确也不会报错,而ES5中则会报错.

使用扩展运算符对数组进行赋值
在ES5中如果使用let arr2 = arr1 对数组进行赋值,则是将arr1的内存地址映射给arr2,如果arr1改变,arr2一样会改变.
而在ES6中如果使用…对数进行赋值,则只是将arr1的值arr2,等于创建了一个新的arr2数组.

rest运算符(剩余部分) ‘…’

rest和扩展运算符都是’…’用法几乎相同,但是rest有剩余的意思,它只会算通过…传入的参数部分,通过直接传入的不会算进去

使用for of循环打印数组

使用for of替换for 循环可以避免在内存中开辟空间来存索引i
let arr = [1,2,3]; for(val of arr) {console.log(val)//1,2,3}

四.字符串模板

ES5中使用’+’来拼接字符串,而在 ES6中要使用`${字符串}`和来配合,这就是字符串模板并且可以在里面嵌入html标签

字符串模板也支持运算


字符串查找 被搜索字符串.includes(要搜索字符串)
查找开头有没有 被搜索.startsWith(要搜索)
结尾有没有被搜索.endsWith(要搜索)
字符串复制 要被复制.repeat(次数)

五.数字的操作

二进制和八进制数

二级制Binary
声明二进制用 0B开头
let a = 0B1010101

声明八进制

let b = 0o666;

数字判断

判断是否为数字 Number.isFinite(要判断值)

判断是否为NaN Number.isNaN(要判断值)

Number.isInteger(要判断值)

整型浮点转换 Number.parseInt() NumberparseFloat()

取值范围

最大取值是2*53-1次方
最小取值是-2*53+1次方

最大安全整数,最小安全整数常量
Number.MAX_SAFE_INTEGER
Number.MIN_SAFE_INTEGER

判断是否是安全整数数值范围
Number.isSafeInteger()

六.ES6数组扩展知识

JSON转换成数组 Array.from(json)

定义一个数组形式的json
let json = {"0":"luan","1":"hao","2":"chen",lenght:3}
转换一个数组形式的json
let arr = Array.from(json); console.log(arr);

将各种数据转换成数组格式 Array.of(数据)

let arr = Array.of('abcd') ; let arr = Array.of("a","b","c","d")

find() 实例方法

方法接收一个回调函数,回调函数接收三个参数,value表示当前查找的值,index表示查找的索引,arr表示查找的数组,如果找不到则返回undefined
let arr = [1,2,3,4]; arr.find(function(val,index,arr){ return val>2})

fill()实例方法

将数组进行填充,接收三个参数,第一个是替换变量,第二个是开始填充的位置,第三个是填充到的位置,不包括结尾

for of循环

for(let val of arr)
arr.keys()输出数组的下标
arr.entries()输出数组加下标 

    var arr =  [a,b,c]
for(let [val,i] of arr.entries()) {
    console.log(i+'...'+val)//0...a 1...b 2...c
}

entries()方法

可以将数组生成条目形式的数组,可以通过next跳到下一条条目,可以再格式不规则的情况下输出数组的值.

    let arr = [a,b,c];
    let list = arr.entries();
    console.log(list.next().value)//a
    console.log(list.next().value)//b
    console.log(list.next().value)//c
