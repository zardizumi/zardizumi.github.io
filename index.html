static关键字

类变量和类方法
如果我们希望即使该类有很多的实例，但是某些数据在空间中只有一份，而不是每一个实例都有一份，就可以使用类变量和类方法。
即所有该类的实例都拥有这些属性方法。
当其中一个对象对静态属性进行修改会导致所有实例的该属性改变。
使用static修饰的属性和方法在内存中存在于静态域，不同于堆和栈空间
类变量随类加载而加载，类变量早于实例变量。可以直接使用类调用。
类方法也是随类加载而加载，可以直接通过类来调用。
静态方法只能调用本类的静态属性和其他静态方法。因为对象加载晚于类加载，所以掉不了非静态的
非静态方法可以调用静态方法和静态属性
静态结构出生早于非静态结构，同时被回收也晚于非静态结构。
在静态方法中不能使用this和super，因为此时对象还没有出现。
static 可以用来修饰属性，方法，代码块，初始化块，内部类

class SportMan {
    String name ;
    int age;
    static String nation;
    public SportMan(String name,int age){
        this.name = name;
        this.age = age;
    }
    @Override
    public String toString() {
        return "SportMan [name=" + name + ", age=" + age + ", nation=" + nation + "]";
    }    
}

单例设计模式

解决的问题,是的一个类只能创建一个对象

  1. 私有化构造器,使得在类的外部不能调用构造器.
  2. 在类的内部创一个类的实例.
  3. 私有化此实例,并通过公共方法来返回

  4. 通过该类调用此方法,所以将此方法设置为static,并将返回的实例设置为static

    ##饿汉式
    package singleton;

    public class Test_1 {

    public static void main(String[] args) {
    Singleton s1  = Singleton.getInstance();
    Singleton s2  = Singleton.getInstance();
    System.out.println(s1==s2);

}

    }

    class Singleton{

    private Singleton(){}
private static Singleton instance = new Singleton();
static Singleton getInstance(){
    return instance;
}

    }

  1. 饿汉一开始就创建好了.

Nuxt.js

该框架可以将vue文件通过SSR经过服务器渲染成html文件返回给服务器,对seo很友好

Nuxt 常用配置项

配置图片的打包

在nuxt.config.js中配置build选项中的loaders

nuxt 路由和通过路由传递参数

只要我们在page目录下新建了vue文件,nuxt会默认为我们加上该vue名字的路由.不需要配置.路由跳转需要使用标签,绑定to属性为一个对象,传入name值为路由,params为参数对象
在页面的vue文件中使用$route.params.参数名来接受传过来的参数

动态路由 下划线.vue文件

Nuxt为我们提供了动态路由,即根据传入路径参数的不同显示不同的页面
所有动态路由都是 _XXX.vue文件
并且Nuxt为我们提供了validate方法,用于校验路由参数的有效性,该方法返回一个boolean,如果返回false则显示Nuxt的404页面

路由动画

全局路由动画

1.在assets文件夹下新建css文件夹并新建css文件,设置路由动画样式,通过nuxt.config配置css.如果想让所有路由跳转都有动画,那么需要通过nuxt-link来跳转

单独设置路由动画

即针对一个某个或某几个路由设置动画,同样在全局css中配置单独的过渡动画,并在该页面的.vue文件中export default的对象中设置transition属性

默认模板

在项目根目录下建立app.html文件即为默认模板,即每一个页面都有相同的公共部分比如title,头部等等

其中nuxt标签为每一页面

设置错误页面

在layouts文件夹下新建error.vue作为出现错误时的页面.

设置页面特有的title和meta

如果我们要设置每一个页面特有的title和meta需要配置页面的head()方法,返回一个对象
